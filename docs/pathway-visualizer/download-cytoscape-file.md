### Download Cytoscape File

The **Download Cytoscape File** allows you to export the current netwok as a cytoscape-compatible ```.json``` file. The file can be uploaded to the cytoscape-desktop application for in-depth analysis of networks. Cytoscape is an open source software platform for visualizing complex networks and integrating these with any type of attribute data.



!!! important "Important"
    This option only exports the **nodes and edges currently visible in the viewport**.  It does not export the full model.

!!! tip "Recommendation"
    To export the **entire model**, we recommend selecting all pathways in the dropdown, applying any changes, and then downloading the tool file.
